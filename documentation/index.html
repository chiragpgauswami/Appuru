Introduction

Appuru is a Bootstrap 5 HTML app landing template with a color switcher feature. It's lightweight and beautiful, with a clean and modern design that's easy to customize. The color switcher feature lets users change the look and feel of the template with a single click. Appuru comes with a variety of pre-built sections, such as a hero section, a features section, a call to action section, and more. It's easy to use and customize with clean and well-organized code.

Designing Files

  • CSS

    • Files are explained below:

    File Description
    vendor This folder is containing all vendor styles.
    style.css This file is containing all common styles for all the pages.
    responsive.css This file is containing all responsive styles for all the pages.
    blog.css This file is containing all styles for the blog pages.
  • JS

    • Files are explained below:

    File Description
    vendor This folder is containing all vendor js.
    app.js This file is containing all common js for all the pages.
